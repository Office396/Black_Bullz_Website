## **App Directory (Main Pages)**

- **`app/page.tsx`** - Homepage/landing page of your website
- **`app/layout.tsx`** - Main layout wrapper (header, footer, navigation)
- **`app/globals.css`** - Global styles and CSS variables


### **Game/Software Pages**

- **`app/game/[id]/page.tsx`** - Individual game/software detail page (dynamic route)
- **`app/game/[id]/not-found.tsx`** - 404 page for non-existent games
- **`app/download/[id]/page.tsx`** - Download page for specific items
- **`app/categories/page.tsx`** - Browse games/software by categories
- **`app/latest/page.tsx`** - Shows newest games/software
- **`app/search/page.tsx`** - Search results page
- **`app/search/loading.tsx`** - Loading state for search


### **Other Pages**

- **`app/contact/page.tsx`** - Contact form/information page


### **Admin Section**

- **`app/admin/portal/page.tsx`** - Admin dashboard/login area


## **Components Directory (Reusable UI)**

### **Main Components**

- **`header.tsx`** - Website header/navigation bar
- **`sidebar.tsx`** - Side navigation menu
- **`game-grid.tsx`** - Grid layout for displaying games/software
- **`game-details.tsx`** - Detailed view component for individual items
- **`search-results.tsx`** - Search results display component
- **`comments.tsx`** - Comments system for games/software


### ️ **Admin Components**

- **`admin-dashboard.tsx`** - Admin control panel
- **`admin-login.tsx`** - Admin authentication form
- **`admin-item-form.tsx`** - Form to add/edit games/software
- **`admin-item-list.tsx`** - List view of all items in admin
- **`admin-settings.tsx`** - Admin configuration settings
- **`admin-download-pages.tsx`** - Manage download pages


### **Loading Components**

- **`loading-screen.tsx`** - Full page loading screen
- **`loading-spinner.tsx`** - Small loading spinner
- **`page-loader.tsx`** - Page transition loader


### **UI Components**(`components/ui/`)

Pre-built UI components like buttons, cards, forms, etc. (shadcn/ui library)

## **Public Directory (Assets)**

- `minecraft-game-cover.png`, `gta-v-game-cover.jpg`, etc.

- **`bull-logo.png`** - Your website logo

## **Other Important Files**

- **`lib/utils.ts`** - Utility functions
- **`hooks/`** - Custom React hooks
- **Configuration files**: `next.config.mjs`, `tsconfig.json`, `package.json`


## **What to Edit for Different Changes:**

**Homepage changes** → `app/page.tsx`
**Header/Navigation** → `components/header.tsx`
**Game listings** → `components/game-grid.tsx`
**Individual game pages** → `app/game/[id]/page.tsx`
**Search functionality** → `app/search/page.tsx` + `components/search-results.tsx`
**Admin panel** → Files in `components/admin-*.tsx`
**Styling** → `app/globals.css`
**Add new pages** → Create new files in `app/` directory
